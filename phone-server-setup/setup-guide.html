📱 Old Phone Setup

LearnoStream Phone Server

Turn your old Android phone into a live streaming server in 2 minutes.

  1. Install Termux on your old phone (free, from F-Droid — not Play Store)

    👉 Download Termux from F-Droid

  2. Open Termux and paste this single command:

    curl -sL https://raw.githubusercontent.com/YOUR_REPO/main/phone-server/install.sh | bash

    ⏳ This takes about 2–3 minutes. It installs everything automatically.

  3. After setup completes, start the server:

    bash ~/learnostream/start.sh

    First time: it will ask for a free ngrok token (30 second signup). Then it shows you a URL like wss://abc123.ngrok.io

  4. Copy that WSS URL into your Vercel project:

    Vercel → Your Project → Settings → Environment Variables
    Add: VITE_PHONE_SERVER_URL = the wss:// URL
    Then click Redeploy.

  5. Done! Go to your site → Admin → Go Live 🔴

Every time you want to stream: Open Termux → run bash ~/learnostream/start.sh — the URL changes each time (free ngrok), so update Vercel env var and redeploy.

Tip: Get a paid ngrok plan ($8/mo) or use a fixed domain to avoid updating the URL every time.
